Sarah Drasner

Sr. Cloud Developer Microsoft

Sarah is an award-winning Speaker, Sr Cloud Developer Advocate at Microsoft, and Staff Writer at CSS-Tricks. Sarah is also the co-founder of Web Animation Workshops, with Val Head. She's the author of SVG Animations from OíReilly and has given Front-end Masters workshops on Vue.js and Advanced SVG Animations. Sarah is formerly Manager of UX Design & Engineering at Trulia (Zillow). Sarah won CSS Dev Conf's "Best of the Best Award" as well as "Best Code Wrangler" from CSS Design Awards.
